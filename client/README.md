# English Master — Client

Ứng dụng web học tiếng Anh, xây dựng với [Next.js](https://nextjs.org) (App Router), React 19 và TypeScript.

## Yêu cầu

- [Node.js](https://nodejs.org/) (khuyến nghị LTS)
- [Yarn](https://yarnpkg.com/) classic (v1)

## Cài đặt

Trong thư mục `client`:

```bash
yarn install
yarn dev
yarn build
